# The purpose of the claw

It should be able to hold on to objects with a good grip. It should be able to have sufficient enough force to hold heavy objects, but also be precise enough to grab flimsy objects like a cloth or clothes. It should have sensors built into it 
to know whether to put a lot of force or not a lot of force into an object. It should know when the fingers are pressing down hard enough onto the object so that it can be lifted
